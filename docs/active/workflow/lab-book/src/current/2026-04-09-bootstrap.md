# 2026-04-09 Workflow Bootstrap

## What Changed

Created the initial formal workflow scaffolding for eval-driven development:

- durable workflow docs under `docs/workflow/`
- living artifacts under `docs/active/workflow/`
- repo-local workflow skills under `docs/workflow/skills/`

## Why

The eval effort is moving from ad hoc prototyping to a repeatable process that can support both engineering decisions and later publication.

Future eval work should have an obvious place for:

- the planned decision
