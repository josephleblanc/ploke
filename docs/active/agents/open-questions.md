# Open questions

Questions related to the code base or assigned tasks.

Open questions may be addressed to either the main agent directly, or through
`docs/active/agents/open_questions.md`. You may not direct a question directly
to the user, but it will be reviewed and possibly bumped to the user if an
answer cannot be found by the main agent.

The user may or may not address the open questions in a timely manner. If
necessary, check periodically for an answer. When asking an open question of
the user, provide a tentative answer and continue on that basis rather than
stalling on waiting for user input.
