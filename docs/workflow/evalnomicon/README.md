# Evalnomicon

Living handbook for eval concepts, introspection methods, review protocols, and
experiment threads.

- book config: [book.toml](book.toml)
- summary: [src/SUMMARY.md](src/SUMMARY.md)

Use this book to synthesize method and framework:

- conceptual foundations belong here
- operational review protocols belong here
- semi-structured experiment and meta-experiment threads belong here

Do not use this book as:

- the live control plane
- the rolling current-state board
- the canonical numeric source for metrics

Those remain under [docs/active/workflow](../../active/workflow).
