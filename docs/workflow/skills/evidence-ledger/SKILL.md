# Evidence Ledger

Update [docs/active/workflow/evidence-ledger.md](/home/brasides/code/ploke/docs/active/workflow/evidence-ledger.md).

## Entry Format

- belief
- evidence
- consequence

## Rules

- Do not log raw narrative here; use the lab book for chronology.
- Do not add a belief without a supporting artifact or reviewed postmortem.
- Revise an existing belief when new evidence sharpens it instead of creating duplicates.
