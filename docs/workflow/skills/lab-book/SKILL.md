## Entry Contents

- what was run

- what we believe now
- next action

## Rules

- Add one dated file per meaningful session or experiment cycle under `src/current/` unless it is already archival.
- Keep chronology here; keep durable decisions in the EDR and durable beliefs in the evidence ledger.
- Move old narrative threads across the archive boundary instead of deleting them.
